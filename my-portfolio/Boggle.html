Boggle

Boggle is a web-based version of the classic word game. The game generates a 4x4 grid of random letters, and players have 90 seconds to find as many words as possible. The game shuffles the board and blurs the letters when the timer runs out.

Play the game!

Features

  • Randomly generated 4x4 grid of letters
  • 90-second timer
  • Shuffle and blur effect when the timer runs out
  • Reset option

Project Highlights

Boggle was a fun project to build because it combined my love for word games with my passion for web development. I enjoyed implementing the timer and shuffle features to create an interactive and engaging game experience.

Technologies Used

  • HTML, CSS, JavaScript
  • React for interactive components
  • CSS for styling
