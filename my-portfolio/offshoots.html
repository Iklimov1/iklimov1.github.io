Offshoots

Offshoots is a dynamic online marketplace for plant cuttings. The platform allows users to browse and purchase various plant cuttings while also providing care instructions and detailed plant information.

Features

  • Intuitive interface for browsing plant cuttings
  • Search and filter functionality for easy navigation
  • Detailed care instructions for each plant
  • User registration, password salting and hashing for security
  • User's can view bids on their plants and place bids on other user's items
  • Real-time sorting of newly added items, organized by price

Project Highlights

One of the standout features of Offshoots is its real-time sorting capability, which allows newly added items to appear immediately without requiring a page refresh. This enhances the user experience by keeping the interface up-to-date dynamically.

Technologies Used

  • HTML, CSS, JavaScript
  • Express.js for backend functionality
  • MongoDB for database management
  • Tailwind CSS for styling
