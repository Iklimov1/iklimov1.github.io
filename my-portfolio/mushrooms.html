Mushrooms of NC

Mushrooms of North Carolina is an interactive website to explore various types of mushrooms found in North Carolina. It includes features like search filters, dark/light mode toggle, and accordion-style information for each species.

Features

  • Dark Mode/Light Mode toggle for a seamless user experience
  • Interactive accordion component for easier navigation of mushroom species
  • Search filter to display mushrooms by name
  • Content-rich interface with easy-to-read species details

Project Highlights

A standout feature of this project is the dynamic interaction with the dark and light mode for improved accessibility. The accordion component enables users to quickly navigate detailed information about each mushroom species.

Technologies Used

  • HTML, CSS, JavaScript
  • Vanilla JS for interactive components
  • MongoDB for database management
  • CSS for styling
